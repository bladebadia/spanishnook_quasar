1 - Ejecucion App:
        A - Abrir vscode (Antes abrir docker)
        B - Actualizar desde git hub: <git pull>
        C - ctrl+shift+p y abrir en contenedor
        D - Abrir terminal en contenedor
        E - ejecutar <npm run dev>
        F - Salir del contenedor a carpeta local
        G - Para guardar cambios en git:
             - git add .
             - git commit -m " texto de la modificacion"
             - git push
